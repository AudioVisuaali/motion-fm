# Motion FM

Plays songs based on movement

## Song locations
```
songs/main
songs/night
```

## Tools

- Rome
- Typescript
- Jest

## Code

### Be sure to install FFmpeg

```
sudo apt update
sudo apt install -y ffmpeg
```

### Install packages
```
npm install
```

### Develop
```
npm run dev
```

### Build
```
npm run build
```
